Wait No More Project is created so that customers for restaurants will be able to sign up for a waitlist. The purpose of this project is for customers to not wait in line, especially during COVID when people don't want to stand close to others for long periods of time. With this project, customers will be able to not wait in line and come when their time is almost up. Restaurants must be registered and staff members will be able to manipulate the waitlist. Link
